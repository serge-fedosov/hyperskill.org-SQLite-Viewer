Description

The main idea of the application is to see the database table contents. So we should be able to find out which tables are included in the database and generate a SELECT query for any table.

Objectives

Add three more components to the application:

  • JComboBox with the TablesComboBox name
  • JTextArea with the QueryTextArea name
  • JButton with the Execute text. Give it the ExecuteQueryButton name.

Don't forget about the components from the previous stage.

Once a user enters the database file name and presses the Open button, establish a connection with the database. Select all public (open) tables from it and put them into the JComboBox. Our app should include an option to select a table and generate a SELECT query for it. This query selects all rows in the table. If a user selects another table in the JComboBox, re-generate a SELECT query for this table.

Here is the query that can be used to get every public table from an SQLite database:

SELECT name FROM sqlite_master WHERE type ='table' AND name NOT LIKE 'sqlite_%';
