Description

Let's create a simple window with a couple of components. In this stage, we need a window that has a text field where we can name a file and a button to establish the connection with our database.

Objectives

Create a window and give it the SQLite Viewer title. The window should contain the following components:

  • JTextField with the FileNameTextField name;
  • JButton with the OpenFileButton name.

Enable the components and make them visible.

To set a name to an element, use the setName() method.
