Description

Let's add an ability to display the data from a table to our application. In this stage, you need to execute a prepared query from the query field that is generated when a user selects a table in the Combobox.

Objectives

Add one more component to the application:

  • JTable with the Table name.

Put this empty table at the beginning of the application. Once a user presses the execute button, take a query from the query field, execute the query, and display the results in the table.
